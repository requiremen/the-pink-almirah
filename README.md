# The Pink Almirah

Premium hand-crafted traditional Indian clothing boutique from the heart of Surat.

## Luxury That Speaks First Before You Do

This repository contains the source code for The Pink Almirah's official website, featuring:
- **Elegant Branding**: Custom logo and typography tailored for a premium boutique.
- **Interactive Menu**: Quick access to our Collections, Store Location, and Instagram.
- **Responsive Design**: Optimized for a seamless experience across all devices.
- **Dynamic Slider**: Showcasing our latest ethnic wear and designer silhouettes.
